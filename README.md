## 📌 Project Overview

Wine quality evaluation is an important task in the food and beverage industry. This project uses Machine Learning to predict wine quality based on physicochemical properties such as acidity, alcohol content, pH, sulphates, and density.

The project demonstrates a complete end-to-end Machine Learning workflow including:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
- Model Comparison
- Hyperparameter Tuning
